🚀 Como Executar
Pré-requisitos
Node.js (para o Angular)

Java JDK / Maven (para o Spring Boot)

Frontend
Navegue até a pasta Front/.

Instale as dependências: npm install

Inicie o servidor: ng serve

Backend
Navegue até a pasta Back/.

Execute o projeto via IDE ou Maven.
